# Smart-Plant-Irrigation-System

###  Project Overview
This project is an Arduino-based smart plant irrigation system designed to automatically water plants when the soil becomes dry. It helps save water and keeps plants healthy without constant manual watering.

### Components Used
- Arduino Uno
- Soil Moisture Sensor
- Water Pump
- Relay Module
- Jumper Wires
- Power Supply

### ⚙️ How It Works
1. The soil moisture sensor measures the moisture level.
2. Arduino reads the sensor value continuously.
3. If the soil becomes dry, the relay turns on the water pump.
4. When the soil reaches the required moisture level, the pump turns off automatically.

### Features
- Automatic plant irrigation
- Water saving
- Easy to use
- Low-cost hardware
- Beginner-friendly Arduino project
